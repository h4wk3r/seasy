# seasy

Simple script to retrieve LINUX basic information

This service is a small utility, its purpose is to centralize all information and display in the terminal.

> CHECKLIST :
- Disk  
- FS
- RAM
- PROCESSOR 
- VERSION 
- INTERFACE

> INSTALL :
- mv seasy.sh seasy
- chmod 755 seasy
- mv seasy /usr/local/bin/
